# Aerial Flow — Сайт студии воздушной гимнастики

Конверсионный сайт для студии воздушной гимнастики для взрослых девушек и женщин.

## Стек

- **Python** + **Flask**
- **HTML** + **CSS** (чистый, без Bootstrap)
- **JavaScript** (чистый, без jQuery)
- **SQLite**

## Как запустить

```bash
# Установить зависимости
pip install flask

# Запустить
python app.py
```

Сайт откроется по адресу: http://127.0.0.1:5000

## Админ-панель

- **URL:** http://127.0.0.1:5000/admin
- **Логин:** `admin`
- **Пароль:** `demo123`

Ссылка "Admin" находится в футере сайта.

## База данных

Проект использует **SQLite**. Файл базы данных `aerial_flow.db` создаётся **автоматически** при первом запуске:
```bash
python app.py
```

База содержит таблицы:
- **leads** — заявки с формы записи
- **events** — события на главной странице
- **news** — новости для админки

При первом запуске в базу добавляются демо-данные (3 события, 3 новости).

**Важно:** база данных не загружается на GitHub (добавлена в `.gitignore`), так как может содержать персональные данные из заявок. При клонировании репозитория запустите `python app.py` — база создастся автоматически.

## Что уже реализовано

### На главной странице
- Hero-блок с призывом к действию
- Блок "Кому подойдёт" — 4 карточки
- Блок "Форматы занятий" — 4 направления с уровнями
- **Мини-тест** на JavaScript — 3 вопроса, персонализированный результат
- Блок "Ближайшие события" — данные из БД
- Блок "Истории учениц" — отзывы
- **FAQ** с раскрывающимися ответами
- **Форма заявки** с сохранением в SQLite
- **Плавающая кнопка** "Записаться" на всех страницах
- Адаптивный дизайн для телефонов и планшетов

### В админ-панели
- **Dashboard** со статистикой
- **Заявки** — таблица со статусами (новая / в работе / обработана)
- **События** — добавление, редактирование, удаление
- **Новости** — добавление, редактирование, удаление, смена статуса
- **Генерация текста поста** для соцсетей по шаблону

### Дизайн
- Розово-сиреневая цветовая гамма
- Glassmorphism (стеклянные карточки)
- Градиентный фон с плавающими декоративными элементами
- Плавные hover-эффекты и анимации
- Женственный, воздушный стиль
