# Aerial Flow — Сайт студии воздушной гимнастики

Конверсионный сайт для студии воздушной гимнастики для взрослых девушек и женщин.

## Стек

- **Python** + **Flask**
- **HTML** + **CSS** (чистый, без Bootstrap)
- **JavaScript** (чистый, без jQuery)
- **SQLite**
- **OpenAI API** (генератор текстов)
- **VK API** (публикация постов в сообщество ВК)

## Как запустить

```bash
# Установить зависимости
pip install flask openai python-dotenv

# Настроить API-ключи (опционально)
# Создайте файл .env в корне проекта и добавьте нужные ключи:
# OPENAI_API_KEY=your_openai_api_key_here
# VK_ACCESS_TOKEN=your_vk_access_token_here
# VK_GROUP_ID=your_vk_group_id_here
# VK_API_VERSION=5.199

# Запустить
python app.py
```

Сайт откроется по адресу: http://127.0.0.1:5000

## Публикация во ВКонтакте

Проект поддерживает публикацию подготовленных постов на стену сообщества ВК через VK API.

### Настройка

1. Создайте токен сообщества ВК с правом `wall` в настройках сообщества → Работа с API.
2. Скопируйте ID сообщества (число из адресной строки или из раздела "Работа с API").
3. Добавьте в файл `.env` в корне проекта:

```env
VK_ACCESS_TOKEN=ваш_токен_сообщества
VK_GROUP_ID=123456789
VK_API_VERSION=5.199
```

- `VK_ACCESS_TOKEN` — токен сообщества с доступом к стене
- `VK_GROUP_ID` — числовой ID сообщества (без минуса)
- `VK_API_VERSION` — версия API (по умолчанию 5.199)

### Как работает

1. Создайте или сгенерируйте текст поста в разделе «План публикаций».
2. Откройте редактор поста.
3. Нажмите «Опубликовать во ВКонтакте».
4. При успехе статус поста меняется на «Опубликован», сохраняется `vk_post_id`.
5. При ошибке сообщение об ошибке отображается в редакторе без раскрытия токена.

### Безопасность

- Токен и ID сообщества хранятся только в `.env`.
- `.env` находится в `.gitignore` и не попадает на GitHub.
- Токен не выводится в консоль, HTML, логи или README.
- При отсутствии настроек ВК кнопка публикации показывает понятную ошибку без раскрытия данных.

## База данных

Проект использует **SQLite**. Файл базы данных `aerial_flow.db` создаётся **автоматически** при первом запуске:
```bash
python app.py
```

База содержит таблицы:
- **leads** — заявки с формы записи
- **events** — события на главной странице
- **news** — новости для админки

При первом запуске в базу добавляются демо-данные (3 события, 3 новости).

**Важно:** база данных не загружается на GitHub (добавлена в `.gitignore`), так как может содержать персональные данные из заявок. При клонировании репозитория запустите `python app.py` — база создастся автоматически.

## Что уже реализовано

### На главной странице
- Hero-блок с призывом к действию
- Блок "Кому подойдёт" — 4 карточки
- Блок "Форматы занятий" — 4 направления с уровнями
- **Мини-тест** на JavaScript — 3 вопроса, персонализированный результат
- Блок "Ближайшие события" — данные из БД
- Блок "Истории учениц" — отзывы
- **FAQ** с раскрывающимися ответами
- **Форма заявки** с сохранением в SQLite
- **Плавающая кнопка** "Записаться" на всех страницах
- Адаптивный дизайн для телефонов и планшетов

### В админ-панели
- **Dashboard** со статистикой
- **Заявки** — таблица со статусами (новая / в работе / обработана)
- **События** — добавление, редактирование, удаление
- **Новости** — добавление, редактирование, удаление, смена статуса
- **Генерация текста поста** для соцсетей по шаблону

### В AI-генераторе текстов
- Выбор типа текста (пост, сторис, анонс, описание, ответ клиенту, новость)
- Настройка темы, формата, аудитории, тональности, длины и призыва к действию
- Генерация через OpenAI API (модель gpt-4o-mini)
- Сохранение всех сгенерированных текстов в базу данных
- История текстов с возможностью копирования и удаления

### В плане публикаций
- Список постов со статусами (черновик, готов, запланирован, опубликован)
- Редактор поста: текст, платформа, статус, дата, заметка, счётчик символов
- Публикация во ВКонтакте через VK API (метод wall.post)
- Сохранение vk_post_id, даты публикации и ошибок публикации

### Дизайн
- Розово-сиреневая цветовая гамма
- Glassmorphism (стеклянные карточки)
- Градиентный фон с плавающими декоративными элементами
- Плавные hover-эффекты и анимации
- Женственный, воздушный стиль
